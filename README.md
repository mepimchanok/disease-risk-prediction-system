# 🩺 Disease Risk Prediction System

A machine learning web application that predicts the **risk of Diabetes and Heart Disease** using both **Ensemble Machine Learning** and **Neural Network models**.

The system allows users to explore the modeling process, compare model performance, and test predictions through an interactive **Streamlit web interface**.

# 📌 Project Overview

This project was developed to explore machine learning approaches for **medical risk prediction** using structured healthcare datasets.

Two different modeling strategies were implemented and compared:

### 1️⃣ Ensemble Machine Learning
Combines multiple classical machine learning models:

- Logistic Regression
- Random Forest
- Gradient Boosting

These models are combined using a **Soft Voting Ensemble** to improve prediction performance.

### 2️⃣ Neural Network
A **Feedforward Neural Network** implemented using **TensorFlow / Keras**.

# 📊 Features of the Web Application

The web interface includes several sections:

### 🧠 ML Approach
Explains the machine learning workflow including:

- Dataset preparation
- Feature engineering
- Model training
- Ensemble learning

---

### 🤖 Neural Network Approach
Describes the neural network architecture and training process.

---

### 🧪 Test ML Model
Allows users to input medical features and test predictions using the **Ensemble Machine Learning model**.

---

### 🧪 Test NN Model
Allows users to test predictions using the **Neural Network model**.

---

### 📊 Model Comparison
Compares both models using multiple evaluation metrics:

- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC
- Confusion Matrix
- ROC Curve

The system also recommends the **best model based on F1-score**.
